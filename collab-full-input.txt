## Signal Theory Framework (MASTER REFERENCE)

### The Four Signal States

**1. Interest**
- **What it looks like:** Reciprocal effort, consistent forward movement, she initiates contact, suggests specific plans, follows through
- **How to respond:** Stay steady, match investment, don't accelerate
- **Common mistake:** Over-investing because you're excited

**2. Uncertainty**
- **What it looks like:** Inconsistent initiation, warm but distant after, vague plans, fluctuating energy
- **How to respond:** Stay warm but not insistent, give space, don't push for answers
- **Common mistake:** Trying to "fix" uncertainty with more contact or explanations

**3. Comfort**
- **What it looks like:** Easy conversation, emotional openness, no romantic direction, no escalation
- **How to respond:** Reintroduce direction calmly or step back
- **Common mistake:** Lingering indefinitely hoping attraction will emerge

**4. Disengagement**
- **What it looks like:** Moving away, slower responses, plans falling through, fading energy
- **How to respond:** Accept direction, stop pushing, preserve dignity
- **Common mistake:** Interpreting disengagement as uncertainty and trying to win her back

### Core Framework Rules

1. **Signals Over Stories** - Trust behavior over explanations. When words and behavior conflict, behavior wins.
2. **Timing Over Intent** - When you do something matters more than why. Kind intentions early create pressure.
3. **Calibration Over Performance** - Match energy to situation. Don't try to impress or force connection.
4. **The Master Question** - "Am I aligned with the signal or trying to change it?" If trying to change it, stop.

### Results Screen Updates

**Add state-by-state breakdown:**

```javascript
function showQuizResults() {
  // ... existing score calculation ...
  
  // ADD: State-by-state breakdown
  const stateBreakdown = {
    Interest: { correct: 0, total: 0 },
    Uncertainty: { correct: 0, total: 0 },
    Comfort: { correct: 0, total: 0 },
    Disengagement: { correct: 0, total: 0 }
  };
  
  quizState.scenarios.forEach((q, i) => {
    const answer = quizState.answers[i];
    stateBreakdown[q.signalState].total++;
    if (answer?.correct) {
      stateBreakdown[q.signalState].correct++;
    }
  });
  
  // Display breakdown
  let breakdownHTML = '<div class="state-breakdown">';
  breakdownHTML += '<h3>Your Signal State Recognition:</h3>';
  
  Object.entries(stateBreakdown).forEach(([state, stats]) => {
    if (stats.total > 0) {
      const icon = stats.correct === stats.total ? '✅' : '⚠️';
      breakdownHTML += `<div class="state-stat">${icon} ${state}: ${stats.correct}/${stats.total} correct</div>`;
    }
  });
  
  breakdownHTML += '</div>';
  
  // Identify struggling areas
  const struggling = Object.entries(stateBreakdown)
    .filter(([_, stats]) => stats.total > 0 && stats.correct / stats.total < 0.7)
    .map(([state, _]) => state);
  
  if (struggling.length > 0) {
    breakdownHTML += `<div class="struggling-note">You Struggled With: 🎯 ${struggling.join(', ')}</div>`;
  }
  
  document.getElementById('resultsBreakdown').innerHTML = breakdownHTML;
}
```
