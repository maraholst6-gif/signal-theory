# Content Gating Strategy — All 8 Profiles

**Principle:** Free content = diagnosis (what's wrong). Gated content = prescription (what to do). Page unlocks remaining analysis. Email delivers the practices.

---

## Universal Page Structure

```
1. Header (profile name + tagline)
2. 3 Scores with brief descriptions
3. Brief interpretation (1-2 paragraphs)
4. First pattern/blind spot (teaser)
5. ⛔ EMAIL GATE
6. [UNLOCKS] Remaining patterns + consequences
7. App/book bridge + social share
```

**Email delivers:** 3 profile-specific practices (NOT on the page)
**Email includes:** Link back to full results page
